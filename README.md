# 运维开发管理系统

使用 pip install -r requirements.txt 安装包依赖；

进入项目目录，运行 python manage.py runserver ； 默认端口 8000;

进入网站: http://localhost:8000

进入管理后台: http://localhost:8000/backend/

管理员用户员/密码 ： admin/ admin
